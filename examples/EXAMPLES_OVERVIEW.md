## Design Patterns Worth Adopting

### Visual Hierarchy
- **Gradient accent colors** for emphasis (see examples)
- **Card-based layouts** for scanability
- **Layered architecture diagrams** for system thinking
- **Hover states** showing interactivity

### Content Structure
- **Challenge → Approach → Outcome** for projects
- **Takeaway boxes** for key insights
- **Trajectory sections** for forward-looking analysis
- **Tech stack badges** for credibility

### Navigation Patterns
- **Hub-and-spoke** (Research Hub as central manifesto, essays/projects as spokes)
- **Category filtering** for essays
- **Related content** footers on essays
- **Breadcrumb navigation** for orientation

### Typography
- **Readable line length** (740px max for articles)
- **Generous line-height** (1.6-1.8 for body text)
- **System fonts** for fast loading
- **Scale contrast** between headers and body
